About the App:

This project is my first capstone for the Springboard Software Engineering Program. It aims to find public transit routes within a 500 meter circular radius of an address that a user gives. From there, it finds stations that are located within that 500 meter radius (if any). If you click a station, it will render a map of the next five departures from that station, as well as give some information about each departure. The map, while fairly accurate, is not intended to be read as the exact location of the stop. Rather, it will give you a general idea as to where the route would take you.
